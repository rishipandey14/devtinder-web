## Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm

### Installation

```bash
git clone https://github.com/rishipandey14/devtinder-web.git
cd devtinder-web
npm install
```

### Running the App

```bash
npm run dev
```

The app will be available at [http://localhost:5173](http://localhost:5173).

### Build for Production

```bash
npm run build
```

### Lint

```bash
npm run lint
```
